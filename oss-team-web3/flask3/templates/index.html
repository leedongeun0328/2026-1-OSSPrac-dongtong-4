프로젝트 소개

본 프로젝트는 오픈소스소프트웨어 실습 dongtong 4팀의 팀원 소개 웹사이트를 만들어보는 것이며, 웹 개발 기초와 오픈소스 협업 방식을 팀원들과 함께 실습하는 것을 목표로 합니다. Python Flask를 기반으로 웹 애플리케이션 구조를 만들고, HTML, CSS, Bootstrap을 활용하여 팀원 소개 페이지를 구성해보았습니다. 각 팀원의 역할과 기술 역량을 소개할 수 있도록 화면을 설계하였으며, GitHub의 Fork, Branch, Pull Request, Merge 과정을 활용하여 팀원 간 협업을 통해 진행 되었습니다.

역할 분담

팀원별로 페이지 구성, 디자인 수정, Flask 기능 구현, GitHub 협업 과정을 분담하여 진행합니다.

기술 스택

Python Flask를 기반으로 웹 구조를 만들고, HTML, CSS, Bootstrap을 활용하여 반응형 화면을 구성합니다.

오픈소스 협업

GitHub Fork, Branch, Pull Request, Merge 과정을 활용하여 팀원 간 협업 흐름을 실습합니다.

Flask 기반 웹 애플리케이션 구조

User 웹사이트 접속
Browser 페이지 요청
Flask 요청 처리
Templates / Static HTML, CSS, 이미지 파일 구성
Bootstrap 반응형 화면 구현

사용자는 브라우저를 통해 웹사이트에 접속하고, 브라우저는 Flask 애플리케이션에 페이지를 요청합니다. Flask는 요청에 맞는 HTML 템플릿을 렌더링하고, static 폴더의 CSS와 이미지 파일을 함께 불러와 화면을 구성합니다. 또한 Bootstrap을 활용하여 다양한 화면 크기에서도 보기 좋은 반응형 페이지를 구현하였습니다.
