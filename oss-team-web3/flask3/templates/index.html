프로젝트 소개

본 웹사이트는 4팀 팀원 소개를 위해 만들어진 프로젝트로, 팀원들의 역할과 기술 역량을 소개하기 위해 제작되었습니다. Python Flask를 활용하여 웹 애플리케이션의 기본 구조를 구성하고, 수업에서 구현한 Bootstrap을 이용하여 반응형 화면 만들었습니다. 또한 Nginx와 Gunicorn을 연동하여 운영 서버 환경에서도 안정적으로 서비스할 수 있는 구조를 목표로 합니다.

역할 분담

백엔드, 프론트엔드, 배포 담당을 명확히 구분합니다.

기술 스택

Python, Flask, Bootstrap, Nginx 등 주요 기술을 시각화합니다.

오픈소스 협업

GitHub 기반 협업과 오픈소스 도구 활용 경험을 담습니다.

오픈소스 사용 기술 아키텍처

User Browser
HTML / CSS Bootstrap UI
Nginx Reverse Proxy
Gunicorn WSGI Server
Flask Python App

사용자는 브라우저를 통해 웹 페이지에 접속하고, Nginx는 외부 요청을 받아 Gunicorn으로 전달합니다. Gunicorn은 Flask 애플리케이션을 실행하며, Flask는 HTML 템플릿과 정적 파일을 이용해 사용자 화면을 구성합니다.
